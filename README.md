# Portfolio Website

Hi, I’m Ally  
This is my personal portfolio website built to showcase my projects and experience in software development.

---

## About

This project is a fully custom website built from scratch using HTML, CSS, and JavaScript.  
The goal was to create a clean, interactive space to present my work in a way that actually feels like me.

---

## Tech Stack

- HTML  
- CSS  
- JavaScript  
- Git and GitHub  

---

## Features

- Responsive layout for different screen sizes  
- Interactive project gallery  
- Media support for images and videos  
- Resume preview and download  
- Contact form with basic validation  

---

## Purpose

This project helped me strengthen my front end skills and better understand how to structure and design a full website from start to finish.

---

## Running Locally (for now)

1. Clone the repository  
2. Open the project folder  
3. Run with Live Server or open `index.html` in your browser
