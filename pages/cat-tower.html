Cat Tower

Overview

This project focuses on 3D modeling and asset creation in Maya, where I designed and built a detailed cat tower from scratch. The goal was to create a clean, game ready model while exploring structure, proportions, and how real world objects translate into a digital environment. I focused on making the asset feel believable and functional, thinking through how each part of the tower would realistically be used.


Throughout the process, I worked on refining both the visual quality and the technical side of the model, paying attention to details like shape consistency, surface elements, and overall composition. This project helped me better understand how to balance aesthetics with performance, especially when preparing assets for real time use.


Beyond modeling, this project is also a step toward animation and game integration. The cat tower is being prepared for use in Unreal Engine, where it will be used to explore animation, interaction, and how assets exist within a larger environment. As a work in progress, it reflects my growth in 3D modeling while also marking my transition into animation and real time workflows.

Tools

  • Maya
  • Unreal Engine 5

Process

The development process started with blocking out the main structure of the cat tower in Maya to establish the overall proportions and scale. I focused on building a strong base first, making sure each level of the tower felt balanced and believable. From there, I gradually refined the model by adding platforms, supports, and smaller structural details, thinking through how the object would exist and function in a real environment.


A big part of the process involved experimenting with different modeling techniques to create more complex shapes, especially for elements like cylindrical supports and layered surfaces. I worked through challenges like shaping curved forms, adjusting geometry, and keeping everything clean and consistent. This included going back and refining parts of the model multiple times to improve the overall look and fix issues with proportions or alignment.


Once the main structure was complete, I focused on cleaning up the geometry and making the asset more optimized for real time use. This involved organizing the mesh, removing unnecessary geometry, and making sure everything was structured in a way that would translate well into a game engine. I also paid attention to how different parts of the model connected so it would be easier to work with later during animation.


As the project progressed, I started preparing the asset for export into Unreal Engine. This included thinking ahead about how the model would be used, how it could be animated, and how it would fit into a larger environment. The next phase of the project is focused on bringing the model into Unreal, testing it in a real time setting, and experimenting with animation and interaction to make the asset feel more alive and integrated into a game world.
